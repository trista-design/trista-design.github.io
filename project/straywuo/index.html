{ 寵物領養協尋平台 }

Straywuo 浪窩

  • Duration .

    2023/03 - 04

  • Project .

    Web Case Study

  • Role .

    UIUX Designer

是一個寵物領養平台，可以免費刊登動物協尋與認養資訊。此外，平台還提供各種寵物知識文章和推薦獸醫等資訊，讓使用者可以在同一個平台上完成領養後續照護的相關事宜。

{ Background }

近年來，越來越多人關心領養寵物的議題，但許多可愛的毛孩仍未能找到適合的家。儘管有多個領養平台，包括公立收容所和網路上的選擇，但有些人對公立收容所的印象不佳，認為環境可能不夠乾淨和健康。其他平台的領養條件可能過於複雜，讓人感到困惑和挫折，這可能導致他們選擇購買寵物。因此，我們希望建立一個簡單易懂的平台，幫助更多可愛的寵物找到適合的家庭。

{ The Goal }

我們的目標是讓領養寵物變得簡單，協助失踪的毛孩回家，讓毛孩們找到幸福的家，同時分享有價值的知識以及獸醫院評價，希望透過這些努力，能夠減少棄養的情況，讓更多毛孩能夠擁有溫馨的家。

{ Secondary Research }

首先在網上進行研究，收集背景信息，以了解領養動物的現況、領養流程和相關問題。

流浪貓狗數量逐年上升

根據農委會的統計調查，全台遊蕩犬數量逼近十六萬隻，比前兩年多了將近四千隻。這不僅造成生態危機，也讓公立收容所和私人動物收容所面臨收容不足的危機，到了2023年，全台收容所平均容留比已高達近9成，其中包含台北市動物之家在內，一共有8個收容所超額收容。

寵物繁殖業逐年增加

以2021年為例，台灣約有28％的家庭有養寵物，但寵物登記與領養數額約有20萬隻的巨大差距，足見仍有許多寵物犬貓係來自特定寵物繁殖業或民眾家庭自行繁殖者。估計每年會有約16萬隻幼犬被繁殖，但同時卻有超過10萬隻犬貓進入收容所。

領養流程可能過於繁瑣

每個領養平台有自己的一套領養流程，有些可能不夠清楚易懂。人們可能會遇到一隻喜歡的浪浪，但由於領養條件過於繁瑣，這可能會讓潛在的寵物主人感到挫折，導致他們放棄領養的念頭。這樣的情況可能會讓人放棄領養的念頭，這反過來可能會對解決流浪動物的問題造成一定程度的影響。

這個使用者群確認了我們的初始假設，並提供了寶貴的建議，以下是整理這些細節轉化成兩個人物誌。

  • 01Susan Chen | 52 歲 | 退休教師

{ Pain Points }

根據獲得的資料以及使用者訪談提出的痛點與需求，整理出三大重點。

1.

寵物資訊

許多浪浪因送養資訊不足，導致無法提前了解狀況，來評估是否適合領養，使決定變得困難。

2.

領養流程

有時看到心儀的浪浪，但可能因繁複的流程和詢問私人問題讓人感到困擾，最終領養失敗。

3.

寵物護理知識

很多寵物主人不知道如何選食物、應對寵物健康和行為問題，讓他們擔心寵物的健康和幸福。

以上是許多具體的概念和初步草稿，可以有效幫助我快速進入具體可行的設計方案。

{ Mock Up }

在確定設計方向後，我們開始製作高保真原型，展示更多細節和概念，為後續使用性測試做好準備。

主頁面

我們致力於讓一切變得簡單，讓使用者輕鬆取得所需資訊。在我們的平台上，你能輕鬆一站式尋找展覽、藝術家和畫廊的所有資訊！

浪浪認養

可以透過篩選功能，找到最符合使用者生活方式和喜好的寵物。網站提供寵物的詳盡資訊，可以即時聯繫領養機構，詢問領養流程。

獸醫院評價

提供詳細的獸醫院評論，幫助您了解獸醫院的專業水平和服務。可以為您的毛孩選擇最適合的照護，確保牠們獲得最佳的醫療服務。

毛孩知識

透過提供寵物知識，幫助飼主更好地照顧毛孩，希望可以降低遺棄風險，讓他們一同享受美好時光。

{ Learning & Takeaway }

在執行這專案的過程中，我也從中學到了以下幾點：

  • 1.

    ‍設計響應式網站 - 體會到響應式設計在不同設備上，提供一致的使用體驗的重要性。我學習了如何調整內容和布局，以適應不同大小的屏幕，從而確保任何設備上都能提供良好的瀏覽體驗。

  • 2.

    社會影響的力量 - 執行過程中，我意識到即使是一個小網站，也能對社會產生積極影響。我希望透過這個專案能夠鼓勵人們領養寵物、幫助流浪動物找到家，同時減少遺棄情況。這個經驗也讓我深刻了解到，技術和設計能夠為社會帶來正面的改變。
