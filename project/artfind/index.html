{ 尋覓藝文展覽之旅 APP }

Art_Find

  • Duration .

    2023/01 - 02

  • Project .

    App Case Study

  • Role .

    UIUX Designer

是一款整合文化展覽資訊的導覽APP，輕鬆尋找藝術品和展覽的完整資訊，提供個人導覽設備和雲端導覽系統，實現實體與線上完美結合，讓您開啟藝術的奇妙之旅。

{ Background }

平常我喜歡參觀各種藝術展覽，然而漸漸發現，有些有趣的展覽會因缺乏曝光而被忽略。這引起了我的思考：如何找到展覽相關的資訊？如何更有效地計畫參觀行程？當我在展覽中漫遊時，有沒有方法可以更深入地理解每件藝術品的故事？我相信這是提升藝術愛好者在展覽中體驗的一個機會。

{ The Goal }

希望創建一個簡單而貼心的APP，可以輕鬆找到全台各地的藝術畫廊和展覽活動，獲取最新的資訊。並提供便利的導覽工具，讓參訪者更深入地體驗每個藝術作品。通過這個平台，希望能夠重新點燃人們對藝術的熱情，並改善他們在藝術畫廊中的體驗，讓每次參觀都成為一段豐富的故事。

Process
  • Empathy
    我們進行用戶訪談後，理解了問題並定義目標。
  • Ideate
    頭腦風暴發想解決方案，找到最佳的設計方向。
  • Design
    進入設計階段，繪製線框稿與製作高保真原型。
  • Test
    進行測試階段，測試原型、收集用戶反饋。

{ Personas }

為了更深入了解問題範疇，我們與5位對藝術感興趣的參與者進行了非正式訪談。
這個使用者群確認了我們的初始假設，並提供了寶貴的建議，以下是整理這些細節轉化成兩個人物誌。

  • 01 Jennifer | 40 歲 | 會計師
  • 02和丈夫閒暇時喜歡參觀藝術展覽，購買獨特的設計物品。但為了參觀展覽，常常需要花費大量時間去查找資料，他們希望能更方便獲取展覽資訊，享受更豐富的畫廊參觀體驗。
< 我們喜歡參觀藝術展覽，並購買獨特的設計物品。 >
  • 01 Jackson | 25 歲 | 視覺設計師
  • 02對自己的工作充滿熱情，經常參觀展覽來尋找設計靈感。但有時因展覽資訊不足，讓他感到失望，無法真正了解作品。他渴望有辦法可以深入理解每件作品的背後故事。
< 他熱愛從藝術展覽中汲取更多靈感，並與之互動。 >

{ Pain Points }

根據獲得的資料以及使用者訪談提出的痛點與需求，整理出三大重點。

1.

曝光度不夠

許多人因展覽曝光不足而錯過最新活動，也厭倦閱讀冗長的網絡文章，希望能更輕鬆收到資訊。

2.

導覽功能

有時在觀賞藝術作品時，因爲資訊不足，常難以理解藝術家真正的用意，令人感到困惑。

3.

等待時間過長

人們不喜歡排隊等待，這不僅不方便，還浪費寶貴時間，影響了整個參觀體驗的順暢和樂趣。

以上是許多具體的概念和初步草稿，可以有效幫助我快速進入具體可行的設計方案。

{ Mock Up }

在確定設計方向後，我們開始製作高保真原型，展示更多細節和概念，為後續使用性測試做好準備。

主頁面

我們致力於讓一切變得簡單，讓使用者輕鬆取得所需資訊。在我們的平台上，你能輕鬆一站式尋找展覽、藝術家和畫廊的所有資訊！

展覽頁面

收錄了展覽的詳細資訊，使用者可透過評論挑選展覽，買票後更能使用導覽設備預先探索藝術的世界！

導覽頁面

實體與線上的完美串連，無論何時、何地自由走逛各地展覽，你可以自由選擇聽展，或是踏入VR實境的奇幻世界，更深度地感受藝術作品的魅力。

藝術家介紹

使用者可在這裡了解藝術家的設計理念以及其他作品，同時也能直接在應用程式中購買自己喜愛的藝術品。

以上是更多高保真原型設計頁面，我們呈現了對產品不同方面更細緻的設計。

{ Learning & Takeaway }

在執行這專案的過程中，我也從中了解了三件最重要的事情：

  • 1.

    永遠要以使用者為中心思考 - 在設計過程中，始終要把使用者的體驗放在首位，確保他們能夠輕鬆地尋找展覽資訊，並深入參與藝術世界。

  • 2.

    同時解決問題和創造商業價值 - 良好的產品不僅能夠解決使用者的痛點，還能夠在商業層面上有所突破。這個專案讓我明白，平衡這兩者的重要性。

  • 3.

    沒有好只有更好 - 優化是一個持續的過程，隨著使用者的反饋和需求的變化，我們需要不斷地進行改進，確保產品更好地滿足使用者的期望。
